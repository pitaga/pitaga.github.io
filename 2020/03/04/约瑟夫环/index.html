圆圈中最后剩下的数字

来源：leetcode 《剑指Offer（第二版）》 062

问题简述：n个人围成一圈，第一个人从1开始报数，报m的将被淘汰，下一个人接着从1开始报。如此反复，最后剩下一个，求最后的胜利者。

状态表示：f[i][j] 表示当问题规模为1，报数长度为j时最后胜利者的编号。

状态转移：f(i, j) = (f(n - 1, m) + m) % n

初始化：f[1][m] = 0
